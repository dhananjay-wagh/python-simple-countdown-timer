A simple command-line countdown timer built using Python.

This script takes a user-defined duration (in seconds) as input and counts down to zero, pausing one second between each step using the built-in time module. It demonstrates the use of:

range() with step values

reversed() function

time.sleep() for execution delay

Basic user input handling

How It Works

The user enters a countdown duration in seconds.

The program iterates in reverse from the given number down to 0.

Each number is printed with a 1-second delay.

Once complete, it displays "Times up!!".
